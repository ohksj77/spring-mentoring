# spring-mentoring
### 목표 : 간단하게 DB를 활용한 API를 만들며 전체적인 흐름 파악
## 서비스 도메인
- Article (게시글)
## 서비스 요구사항
- [POST] 게시글 작성
- [GET] 단건 조회
- [GET] 전체 조회
- [GET] 타입별 전체 조회
- [POST] 좋아요 추가
- [POST] 좋아요 취소
- [DELETE] 게시글 삭제
## 사용 기술
- Spring MVC
- Spring Data JPA
- Lombok
- H2 Database
## Version
- V1 -> 간단히 구현한 버전
- V2 -> DTO, Mapper, BaseTime, ResponseEntity 등을 추가한 버전
